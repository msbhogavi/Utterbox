# Utterbox

[Live website](https://msbhogavi.github.io/Utterbox/)  
[Download for macOS](https://msbhogavi.github.io/Utterbox/Utterbox.dmg)  
[Source code](https://github.com/msbhogavi/Utterbox)

Utterbox is a camera-aligned macOS teleprompter built for two speaking setups:

- `MacBook Camera`: a sticky notch-aligned prompter for built-in webcam use
- `External Camera`: a floating prompter for external displays and off-center webcams

## Highlights

- Voice follow with privacy-first defaults
- Manual scroll while playing
- Live resize controls
- Menu bar controls and keyboard shortcuts
